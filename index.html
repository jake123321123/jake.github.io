dictionnaires

  1. they are mutable ,iterable and unordered

  2. written between {} with keywords being of string type '' and each set is seperated by (,)

  3. to create a dict we can use multiple methods :
    1. a = dict(p = 4 ,c = 7) => {'p' : 4 , 'c' : 7}
    2. a = dict({'p' : 4},c = 7) => {'p' : 4 , 'c' : 7}
    3. a = dict([(p,7),(c,4)]) => {'p' : 4 , 'c' : 7}
    4. a = [p,c]"list of keys" , b = [4,7]'list of values' --->dic = dict(zip(a,b)) => {'p' : 4 , 'c' : 7}
    5. !!! note : with zip method both arguments need to be of the same length if not use ( zip(list,[None]*len(dic) )

  4. to create a copy of dict we use : a = dict(c) or a= c.copy() or a = c + {}
